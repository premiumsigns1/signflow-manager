## Pricing Reference

### Labour Rates
- **Hourly:** £50-80/hour (covers "lost opportunity cost" for time away from shop)
- **Half day (4 hrs):** £200-320
- **Full day (8 hrs):** £400-640

### Overhead Allocation
- Fixed: ~£33-40/job
- Per job: (Monthly Overheads / Avg Monthly Jobs) + (Job Hours × £10/hr utilities/tools)

### Material Costs (Trade Level, 2026)
- Gloss white cast vinyl (8-year): £5-12/m²
- High-tack printable vinyl: £10-20/m²
- Dibond/ACM panels: £20-50/m²
- Foamex/PVC: £8-20/m²

### Material Formula
- Total m² = (Length_m × Height_m × Qty) × 1.25 (20-30% waste allowance)
- Consumables: 10-20% of material cost + £5-20 fixed

### Markup
- Materials: 3-5x markup (4x for premium)
- Labour: £50-80/hour
- Profit margin target: 40-60% overall

### Quote Formula
```
Total = (Material Cost × Markup_Factor) + (Labour Hours × Hourly Rate) + Overhead Charge + Extras (parking, travel)
```

### VAT
- Always add 20% (UK standard)

### Rounding
- Round totals to nearest £50/£100

## Quoting Process

### Step 1: Receive Job Description
- Identify missing details (size, surface, colours, quantity, turnaround, install vs collection)
- If critical info missing: ask explicit numbered questions
- Don't quote until details clear or assumptions confirmed

### Step 2: Research if Needed
- Use web_search for current material prices
- Use conservative (higher) estimates for safety

### Step 3: Calculate Pricing
- Estimate materials with waste allowance
- Estimate labour (design + production + install + travel)
- Add overhead allocation + profit margin

### Step 4: Prepare Quote
See quote format below.

## Quote Format

```

[Business Name] - Croydon, Greater London
Date: [Current Date in DD/MM/YYYY]

[Brief thanks/acknowledgment]

> [Item 1] (integrate durability, e.g., "premium ... high performance material chosen for excellent UV resistance and long term outdoor performance")
> [Item 2]
> [...]

PRICE BREAKDOWN (EXCLUDING VAT)
> [Description]: £[Amount]
> [Description]: £[Amount]
> TOTAL EXCLUDING VAT: £[Total]
> VAT (20%): £[VAT]
> GRAND TOTAL INCLUDING VAT: £[Grand]

PAYMENT TERMS
> 30 Day Invoice (default; or 50% deposit if large job)

TURNAROUND
> [Timeframe] from [trigger] (subject to scheduling and weather)

[Assumptions note if any]

We're proud of the consistent 5 star feedback from our clients and look forward to delivering the same standard for you.

QUOTE VALID FOR 30 DAYS

Best regards,
[Owner Name]
[Business Name]
[Phone] | [Email]
[Website]
================================================================================
```

### Key Format Rules
- ALWAYS include TOTAL EXCLUDING VAT, VAT (20%), and GRAND TOTAL INCLUDING VAT
- Break out surcharges (urgency, travel, parking) as separate line items in the price breakdown
- Quote validity: 30 days
- Include 5 star feedback line before sign off

## Post-Quote Handling
- If client questions pricing: provide transparent explanation
- Revisions: recalculate if scope changes
- Flag assumptions explicitly to owner

## Example Calculations

### Lettering Install (300-400mm high, 4-6m span)
- Materials: £50-100
- Labour: 6-8 hrs × £60 = £360-480
- Overhead: £150
- **Total: £650-800 ex VAT**

### Formula Application (4.7 m² vinyl)
- Materials: 4.7 × £10 × 4 markup = £188
- Labour: 6 hrs × £60 = £360
- Overhead: £150
- **Total: £698 → round to £700 ex VAT**

### Panels Flip (client delivers, no stripping)
- Original: £1,500 ex VAT (full removal/strip/install)
- Flip: £650 (client delivery, no stripping)
- Savings from eliminated £650 stripping line

### Urgency Surcharge (same day/next day)
- Apply when explicitly requested and confirmed feasible
- State clearly in scope and as separate line item in price breakdown
